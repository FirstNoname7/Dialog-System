1. Создать ноды можно, кликая ПКМ по переговорной или кликая на пробел в переговорной
2. Если у нод одинаковое название, то они подсвечиваются одинаковым цветом (подсказывают, что надо поменять имя)
3. Перемещаться по переговорной можно, зажав колесико мыши
4. Если сверяешься по туторам и видишь, что какие-то моменты не совпадают - у тебя нет группировок нод + ты не модешь выделить несколько нод одновременно, ты решила этот функционал вырезать

Подсказки по ходу создания этой диалоговой системы:
1. Чтобы создать окошко: ПКМ в окне Project->Create->UI Toolkit->Editor Window->оставляешь только C#, у остального галочки снимаешь->Confirm->появится окно и скрипт с логикой создания этого окна.
2. Чтобы создать style sheet (стиль для окошка): сначала создай папку Editor Default Resources (папка с зарезервированным названием для всяких ресурсов дл редакторов кастомных), заходь в неё->ПКМ->Create->UI Toolkit->Style Sheet
3. Как в пару кликов переопределить virtual методы: кликай alt+enter->выбери Generate overrides->выбери методы, которые надо переопределить в данном скрипте
4. Как выделить логику и создать метод одним кликом, поместив туда выделенную логику: выделяй то, что нужно вынести в отдельный метод -> слева появится значок инструмента (быстрые действия) -> Извлечь метод
